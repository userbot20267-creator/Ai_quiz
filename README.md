# 🎓 AI Quiz Bot - أكاديمية الاختبارات الآلية

بوت تيليجرام متكامل لإنشاء وإدارة ونشر الاختبارات بشكل آلي.

## ✨ الميزات

- 📝 إنشاء اختبارات (اختيار من متعدد / صح وخطأ)
- 📤 نشر فوري ومجدول في القنوات والمجموعات
- 🤖 توليد أسئلة بالذكاء الاصطناعي
- 📊 تحليلات متقدمة
- 📅 جدولة النشر (يومي/أسبوعي/شهري)
- 📦 نظام قائمة انتظار
- 🔗 روابط ذكية لكل اختبار
- 🌍 دعم العربية والإنجليزية
- 🔐 نظام حماية متقدم
- 👑 لوحة تحكم للمالك

## 🚀 التشغيل

### المتطلبات
- Python 3.11+
- حساب تيليجرام بوت (من @BotFather)

### التثبيت المحلي

```bash
git clone https://github.com/yourusername/ai-quiz-bot.git
cd ai-quiz-bot
pip install -r requirements.txt
cp .env.example .env
# عدّل ملف .env بالتوكن والإعدادات
python main.py
